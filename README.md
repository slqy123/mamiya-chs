对于语气词，特别是HS中的语气词，翻译起来太折磨了，基本上是摆烂了。大概地修整了一下让它看起来不那么单调，机翻里出来的基本上不是`嗯`就是`啊`，一些特殊的语气词翻译了一下，不影响理解意思的内容就随缘了。
翻译结果方面我很想统一一下但估计出入还是很多，大体的翻译对应关系如下:
```

んく: (嗯)咕
うぅん: 唔嗯
ちゅ: 啾
れろ|あふ: 啊唔
```
